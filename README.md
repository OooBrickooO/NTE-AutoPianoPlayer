NTE AutoPiano (异环自动弹琴)

基于 Python 和 PyQt6 编写的《异环》MIDI 自动弹奏工具。通过底层 DirectInput 模拟键盘输入，安全稳定。

具体使用可看B站视频教程

✨ 功能特点

底层硬件级输入：使用 DirectInput 扫描码，绕过常规游戏屏蔽。

智能化后台控制：支持切出游戏窗口自动暂停，切回自动继续。

动态音域压缩：可强制将超出范围的音符压缩至游戏支持的音域（48~83）。

个性化设置：支持全局速度调节、升降调（-48 ~ +48），支持暗色模式。

进程自动识别：智能下拉框捕捉当前活动进程。

🚀 使用方法

直接运行 (推荐)

前往右侧的 Releases 页面，下载最新的 nte_autopiano_gui.exe 文件。

将下载的 exe 文件放在任意目录。

运行软件，它会在同目录下自动生成 midi_file 文件夹。

将你的 .mid 或 .midi 文件放入该文件夹。

在软件界面点击“刷新”，双击列表中的曲目即可播放。

源码运行

确保安装了 Python 3.8+。

克隆本仓库或下载源码。

安装依赖库：

pip install mido PyQt6 psutil pywin32


运行主程序：

python nte_autopiano_gui.py


⚠️ 注意事项

本软件仅供学习交流使用。

使用 DirectInput 模拟键盘可能会影响你在使用电脑时的其他输入操作，播放时建议保持游戏处于前台。
